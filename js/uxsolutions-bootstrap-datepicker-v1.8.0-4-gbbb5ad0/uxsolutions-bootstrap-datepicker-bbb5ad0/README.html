Development

Once you cloned the repo, you'll need to install grunt and the development dependencies using a package manager:

  • yarn (recommended):
$ [sudo] yarn global add grunt-cli
$ yarn install
  • npm:
$ [sudo] npm install --global grunt-cli
$ npm install
